## How to use this project:
First, you need have virtualenv for running this project ( smooth):

Unix/ Linux
```
pip install virtualenv
```

Create virtualenv:
```
virtualenv env
```

Activate virtualenv:

Windows:
```
.\env\Scripts\activate
```

Unix/ Linux:

```
source env/bin/activate
```

Install packages:
```
pip install -r requirements.txt
``` 

Run this project:
```
python run.py
```
